# Ultra-Sensitive Reverse-Slide Interface — Interactive HTML Dashboard

This is a GitHub-friendly static HTML dashboard based on the provided reference image.

## Run locally

Open `index.html` in a browser.

## GitHub Pages

Upload the unzipped files to a GitHub repository and enable GitHub Pages.

## Included

- Interactive Three.js concept model
- Reverse-slide simulation
- Exploded view
- Cutaway mode
- Wireframe mode
- Live position/force/status indicators
- Engineering charts
- Reference image panel
- Concept OBJ/STL/data/screenshot export buttons

## Safety note

This depicts an inert, non-firing training prop concept. It does not provide firearm construction instructions or manufacturing CAD.
